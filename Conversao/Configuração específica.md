# Configuração específica  
Em alguns formulários de conversão são necessárias configurações específicas. Para isso existe a aba `Configuração Especifica`. Ao contrário da aba de [Configuração Geral](./Configura%C3%A7%C3%A3o%20Geral.md) e das configurações de cada aba, a aba de `Configuração Especifica` varia de formulário para formulário.
